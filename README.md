I'm a Software Engineering undergraduate at **Sabaragamuwa University of Sri Lanka**, focused on building practical, full-stack systems. Most of my recent work has gone into **NearU**, a campus services platform where I designed and built the rides feature end-to-end — backend, real-time communication, and frontend.

- 🎓 Software Engineering student, Sabaragamuwa University of Sri Lanka
- 🔭 Currently building **NearU** — a ride-hailing & campus services platform

- 💻 Full-stack developer — comfortable across backend, frontend, and databases

### 💧 HydroTrack — Water Intake Tracker

A React Native / Expo app that helps users track daily water intake, set goals, and stay on top of hydration with push notification reminders.

**Highlights:**
Daily goal system, persistent local storage with AsyncStorage, and scheduled push notification reminders — built as a hands-on project to learn mobile app development.

`React Native` `Expo` `AsyncStorage` `Push Notifications`
